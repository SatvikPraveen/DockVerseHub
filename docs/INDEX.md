## üéØ Learning Paths

Complete, structured learning paths organized by time commitment and skill level:

### Standard Paths

1. **Beginner Path** - [Complete Guide](learning-paths/beginner-path.md)
   - Duration: 40-60 hours
   - Covers: Docker fundamentals, Compose, basic networking
   - Perfect for: Complete Docker newcomers
   - Includes: 6 hands-on labs with exercises

2. **Intermediate Path** - [Complete Guide](learning-paths/intermediate-path.md)
   - Duration: 50-70 hours
   - Covers: Advanced Docker, Compose, networking, storage
   - Perfect for: Developers familiar with Docker basics
   - Includes: 8 hands-on labs, mini-projects

3. **Advanced Path** - [Complete Guide](learning-paths/advanced-path.md)
   - Duration: 80-120 hours
   - Covers: Orchestration, security, performance, production
   - Perfect for: DevOps engineers, architects
   - Includes: 12 advanced labs, capstone projects

4. **Certification Prep** - [Complete Guide](learning-paths/certification-prep.md)
   - Duration: Variable (20-40 hours)
   - Covers: Exam-specific topics
   - Perfect for: Docker certification candidates
   - Includes: Practice questions, exam strategies

### Time-Constrained Paths

For people with busy schedules - see [Time-Constrained Paths](learning-paths/time-constrained-paths.md)

- **10-Hour Docker Sprint** (2-3 days)
  - Essential Docker knowledge
  - Best for: Busy professionals, quick learning
  - Focus: Hands-on deployment skills

- **20-Hour Weekend Warrior** (1-2 weeks)
  - Fundamentals + intermediate skills
  - Best for: Working professionals
  - Focus: Practical, job-ready skills

- **50-Hour Comprehensive Mastery** (8 weeks or 5-day intensive)
  - Complete Docker expertise
  - Best for: Career advancement, team leads
  - Focus: Deep knowledge + production readiness
